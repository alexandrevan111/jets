## Usage

    jets server

## Description

Runs a local server for development

Starts a local server for development.  The server mimics API Gateway and provides a way to test your app locally without deploying to AWS.

$ jets server

## Options

```
[--port=PORT]          # use PORT
                       # Default: 8888
[--host=HOST]          # listen on HOST
                       # Default: 127.0.0.1
[--noop], [--no-noop]  
```
